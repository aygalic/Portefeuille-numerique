TinyIRC

Chat IRC au design minimaliste

Développé en JavaFX

Application légere et facile d'utilisation

Cette application a été développée dans le but de fournir a n'importe quel utilisateur, un client simple d'utilisation pour accéder rapidement a un serveur de chat IRC. L'IRC est un protocole de communication qui a fait ses preuves et qui est aujourd'hui un choix évident en matière de communication. Nous vous laissons lire la suite de cette documentation technique pour de plus amples informations sur le projet.
Le code source est disponible au téléchargement dans les liens ci dessous. Beaucoup d'informations sur le fonctionnement technique de l'application sont présentent dans la javaDoc alors n'hésitez pas. Nous vous souhaitons une bonne lecture ! :)
