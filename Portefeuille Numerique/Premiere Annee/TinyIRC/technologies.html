Technologies Utilisées

JavaFX 8

API JavaFX basé sur Java 1.8

Cet API permet de créer des interface utilisateur rapidement et facilement.
Il s'adresse aussi bien au développeur qu'au designer et à pour intérêt d'être multi plateforme. En effet ces application peuvent aussi bien être utilisées sur n'importe quel OS de bureau (Windows, Linux, OsX...) que sur des terminaux mobiles ou encore en application web. JavaFX prends aussi en charge le CSS et l'XML qui sont deux langages lié avec l'expérience utilisateur.

CSS3 3

Ce langage est très connu des designer, il permet de mettre en forme les éléments graphiques
Habituellement utilisé avec HTML5, le css permet ici de modifier, d'améliorer l'interface utilisateur en changeant leur propriétés graphiques
