Documentation Utilisateur

Architecture du site

Identification

Pour utiliser l'application web, tout utilisateur doit etre identifié a l'aide d'un login et d'un mot de passe

Menu

Contient les acces au différent affichages, divisé en quatre pages.
Certains auront besoins d'informations pour pouvoir fonctionner, c'est pour ceux la qu'il a une zone de saisie à coté des boutons.
Les titres des sections sont assez explicites

Gerer les utilisateurs

Affiche tout les utilisateur, permet de les supprimer et d'en inserer de nouveaux

gerer les commentaires

Affiche tout les commentaires, permet de les supprimer et d'en poster de nouveau avec votre login

Afficher tout les commentaires à partir d'une date

Grace a la date saisie dans le format donné dans le menu, affiche tout les commentaires postés apres cette date. Il est aussi possible de supprimer les commentaires affichés.

Afficher tout les commentaires d'un utilisateur

Affiche tout les commentaires de l'utilisateur saisi dans le menu. Permet aussi de supprimer les commentaires de cet utilisateur.
