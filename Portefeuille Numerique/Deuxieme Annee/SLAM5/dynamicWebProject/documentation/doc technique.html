Architecture logicielle

Model Vue Controlleur

Model

Contient toutes les classes et les fonctions de l'application. Il est utilisé par les vues pour l'affichage d'éléments de la base de donnée ainsi que le controlleur pour des oppération comme la suppression ou l'ajout d'utilisateur.

Vues

Toutes les pages servant à l'affichage des données, elles font appel au model pour fonctionner et sont appelées par le model

Controlleur

Affiche à l'utilisateur les différentes vues lors de sa navigation. Il fait aussi appel a des fonction du model pour certaines oppération.

Base de donnée

MySQL

Technologie du serveur

JavaEE

L'application ne pourra etre lancée uniquement si vous utilisez un serveur compatible comme par exemple Tomcat ou Tomee

Autre technologies utilisées

Bootstrap

Framework CSS permettant à l'application d'être plus jolie.
